## 🦊 项目简介
这是专为小朋友设计的键盘盲打练习纯前端应用。项目旨在通过有趣、互动的界面，帮助小朋友在轻松愉快的氛围中学习并掌握正确的键盘打字技巧。

**无需安装，无需后端，双击即玩！**

## ✨ 核心特性
- **🎨 趣味化视觉**：色彩鲜艳、大字体设计，吉祥物陪伴练习（火箭 🚀、小狐狸 🦊）。
- **🎹 虚拟键盘反馈**：实时同步真实键盘按键情况，纠正指法。
- **📈 智能进度反馈**：分段练习提示，显示每段完成情况及易错键 Top3。
- **🧠 遗忘曲线复习**：自动生成 D1/D3/D7 复习任务，科学巩固记忆。
- **📊 深度数据看板**：
  - 自动统计 WPM（速度）、准确率、通关星级。
  - **自适应布局**：支持桌面端与移动端切换显示，窄屏下自动切换历史记录与易错排行。
  - **易错键分析**：多维度排序（正确率、错误次数）定位薄弱环节。
- **🏆 成就激励系统**：通关解锁、勋章墙，激发学习动力。
- **🔒 隐私安全**：所有数据均保存在浏览器本地 (localStorage)，无需注册登录。

## 🛠️ 技术栈
- **核心架构**：原生 HTML5, CSS3, JavaScript (ES6+)，**坚持零框架依赖 (Vanilla JS)**。
- **布局方案**：Flexbox & CSS Grid 实现的全响应式设计。
- **存储方案**：Web Storage (localStorage) 实现数据持久化。
- **字体图标**：内置 SVG 图标，确保在各种分辨率下清晰锐利。

## 🚀 快速开始
1. **克隆项目**：
   ```bash
   git clone https://github.com/wangjianghu/keyboard-typing-practice-kids.git
   ```
2. **运行应用**：
   直接双击根目录下的 `index.html`，即可在浏览器中开始练习。

## 📖 练习指南
- **基准键位**：从 F、J 开始，逐步扩展到全键盘。
- **通关目标**：每关都有准确率和速度要求。
- **满星挑战**：追求更高速度与零失误以获取三颗星！
